_axis.offset[3]

Name:
'_axis.offset[3]'

Definition:

        The [3] element of the three-element vector used to specify
               the offset to the base of a rotation or translation axis.

               The vector is specified in millimetres.

Type: float

Mandatory item: no


Enumeration default: 0.0

Category: axis
